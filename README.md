#My ZSH Config

nothing special, i wanna nothing but fast, well, and git prompt

##Install

`git clone https://github.com/zhuangya/zshrc.git ~/.zshrc`

`ln -s ~/.zsh/zshrc ~/.zshrc`

and restart the shell, or `source ~/.zshrc`
